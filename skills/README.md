# Skills — metodik för copy och landningssidor

Den här mappen samlar tre skills som dokumenterar metodiken bakom projektets copy och landningssidor. De är referensmaterial för AI-agenter som arbetar vidare med sajten.

- **copy-alex-hormozi** — bygger oemotståndliga direktrespons-erbjudanden enligt Alex Hormozis $100M Offers-ramverk (rubrik, värdestack, bonusar, garanti, scarcity och CTA).
- **apollo-style-copy** — skriver svensk konverterande copy i Apollos stil för onboarding-/drip-mejl, trial-flöden och webb-copy.
- **saas-landing-architect** — arkitektur för högkonverterande SaaS-landningssidor och marknadssajter (sektionsordning, copy-formler, säljmekanik, layout och svensk anpassning).
